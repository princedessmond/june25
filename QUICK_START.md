# ⚡ QUICK START - Deploy in 20 Minutes

## 🎯 **Goal**: Get your website LIVE with data saving to Google Sheets

---

## **3 Simple Steps**

### ✅ **STEP 1: Google Sheets** (10 min)

1. Create new [Google Spreadsheet](https://sheets.google.com)
2. Make 3 sheets: "Registrations", "Transport Requests", "Legal Aid Requests"
3. Extensions → Apps Script
4. Copy code from `GOOGLE_APPS_SCRIPT.txt`
5. Deploy → New deployment → Web app
6. Copy the URL
7. Paste URL into `google-sheets-config.js` (3 places)

**Done!** ✅ Forms now save to Google Sheets

---

### ✅ **STEP 2: GitHub** (3 min)

1. Create account at [github.com](https://github.com) (if needed)
2. Download [GitHub Desktop](https://desktop.github.com)
3. File → Add Local Repository → Select `/june25/` folder
4. Publish repository (name: `june25-protest-website`)

**Done!** ✅ Code is on GitHub

---

### ✅ **STEP 3: Netlify** (7 min)

1. Sign up at [netlify.com](https://netlify.com) with GitHub
2. Add new site → Import from GitHub
3. Select `june25-protest-website`
4. Click Deploy

**Done!** ✅ **Website is LIVE!**

---

## 🎉 **Your Live URL**

Netlify gives you: `https://yoursite.netlify.app`

**Test it**:
1. Fill out registration form
2. Check Google Sheets → should see your data!
